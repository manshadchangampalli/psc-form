# Claude Code Guidelines

## Project Commands
- Dev Server: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`
- Install: `npm install`

## Standards
- Framework: Next.js (App Router)
- CSS: Tailwind CSS
- Language: TypeScript
- Primary Color: PSC Green (#006837)
- Locale: Malayalam (ml)

## Skill Context
- Use `.agents/skills/` for specific domain knowledge.
- Prioritize Malayalam accessibility and typography.
