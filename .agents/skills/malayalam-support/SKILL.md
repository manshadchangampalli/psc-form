# Malayalam Support Skill

This skill provides guidance on implementing Malayalam text in the web application properly.

## Typography
- Use Google Fonts like **'Meera'**, **'Manjari'**, or **'Gayathri'** for a clean, modern look.
- Ensure `font-display: swap` is used to prevent layout shifts.
- Line height should be slightly larger than English (around 1.6 or 1.8) to accommodate complex characters.

## Text Handling
- Always use UTF-8 encoding.
- When translating marketing copy, prioritize local Kerala PSC terminology (e.g., വില്ലേജ് എക്സ്റ്റൻഷൻ ഓഫീസർ, എൽ.ഡി.സി).
