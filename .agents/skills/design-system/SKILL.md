# PSC Design System Skill

This skill defines the visual identity for the landing page.

## Color Palette
- **Primary Color (Green)**: Use a forest green or a vibrant emerald green that feels professional.
  - Suggestion: `emerald-700` (#047857) or a custom `#006837` (PSC Green).
- **Secondary Color**: Gold or Yellow for "Free Trial" highlights and urgency.
- **Background**: Soft off-white or very light green (#F0FAF4).

## Aesthetic Principles
- **Modern and Premium**: Avoid looking like a generic government site. Use high-quality shadows, rounded corners (`rounded-2xl`), and subtle gradients.
- **Urgency Elements**: Use badges or highlight boxes for the "100 students" limit.
- **Glassmorphism**: Consider using backdrop-blur for the form container to make it feel state-of-the-art.

## Typography
- Pairing a clean sans-serif (Inter/Outfit) for numbers and English headings with a geometric Malayalam font.
